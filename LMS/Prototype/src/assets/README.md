# School Logo Upload Instructions

## Where to Place Your Logo

Place your school logo file in this directory (`src/assets/`).

## Supported Formats
- PNG (recommended for transparency)
- JPG/JPEG
- SVG (best for scalability)

## Recommended Specifications
- **Size**: 200x200 pixels or larger (square format works best)
- **Format**: PNG with transparent background
- **File name**: `school-logo.png` (or .jpg, .svg)

## Example
If your logo file is named `rajasinha-logo.png`, place it here:
`src/assets/rajasinha-logo.png`

The logo will automatically appear in:
- Login page (top center)
- Sidebar navigation (all dashboards)
- Top navbar (optional)
